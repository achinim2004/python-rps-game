# ✂️ Python Rock, Paper, Scissors Game

A simple, console-based implementation of the classic Rock, Paper, Scissors game in Python. Play against the computer!


## 🕹️ How to Play

Enter `1` for Rock, `2` for Paper, or `3` for Scissors when prompted. Enter `4` to quit the game.

## 💡 Key Features

* Uses the `random` module for computer's choice.
* Employs clear `if/elif/else` logic to determine the winner based on game rules.
