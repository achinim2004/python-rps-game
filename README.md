# ✂️ Python Rock, Paper, Scissors Game

A simple, console-based implementation of the classic Rock, Paper, Scissors game in Python. Play against the computer!

---

## 🕹️ How to Play

1.  When the game starts, you will be prompted to make a choice.
2.  Enter `1` for **Rock**, `2` for **Paper**, or `3` for **Scissors**.
3.  Enter `4` to **Quit** the game at any time.
4.  The computer will randomly make its choice, and the winner will be announced instantly!

---

## 🚀 How to Run the Game

Follow these steps to run the game on your local machine:

### Prerequisites
Make sure you have **Python** installed. You can check by typing `python --version` in your terminal.

### Steps to Run:
1.  **Download the Code:** Save the game code into a file named `game.py`.
2.  **Open Terminal:** Open your Terminal or VS Code Terminal (`Ctrl + \``).
3.  **Navigate to Folder:** Use the `cd` command to go to the folder where you saved the file.
4.  **Execute the Script:** Run the following command:
    ```bash
    python game.py
    ```

---

## 💡 Key Features

* **Randomized AI:** Uses Python's `random` module for unpredictable computer moves.
* **Game Logic:** Employs efficient `if/elif/else` conditions and dictionary-based rules.
* **User Friendly:** Includes input validation to handle invalid entries gracefully.
